@title[Getting Started}
### Getting Started
<br>
#### Debian
```shell
~$apt-get install git
```

<br>
#### CentOS
```shell
~$yum install git
```

<br>
#### MacOS
```shell
~$brew install git
```

---
@title[sample code]

### Creating a Repository

```shell
~$mkdir Repository
~$cd Repository
~$git init
Initialized empty Git repository in /home/ben/Repository/.git/
```
